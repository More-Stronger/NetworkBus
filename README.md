### 添加依赖：

#### 1.Add the JitPack repository to your build file
      ```java allprojects {
         repositories {
			   ...
			   maven { url 'https://jitpack.io' }
		   }
	   }
	```
      
#### 2. Add the dependency
	   dependencies {
	          implementation 'com.github.More-Stronger:NetworkBus:1.0.0'
	   }
